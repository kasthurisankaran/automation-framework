# Automation Framework

## 📌 Project Description
This is a Selenium-based automation framework built using C#.  
It follows the Page Object Model (POM) design pattern for better maintainability and reusability.

---

## 🔧 Tech Stack
- Selenium WebDriver
- C#
- MSTest
- WebDriverManager

---

## 🧱 Framework Design
- Page Object Model (POM)
- BaseClass for browser setup
- Config file for environment handling
- Explicit Waits for stability

---

## 🧪 Test Scenarios Covered
- Login functionality with valid credentials

---

## 📁 Project Structure
AutomationFramework/
│
├── Tests/
├── Pages/
├── Utilities/
├── Config/
├── Reports/


---

## 🚀 How to Run the Project
1. Clone the repository
2. Open in Visual Studio
3. Build the solution
4. Run tests using Test Explorer

---

## 📌 Features
- Reusable page methods
- Clean and scalable structure
- Automatic driver management using WebDriverManager
- External configuration using JSON
